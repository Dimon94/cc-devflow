## Character Mapping Rules

| 输入字符类型 | 输出规则 | 示例 |
|-------------|----------|------|
| 中文字符 | 转换为拼音 (pypinyin) | `用户` → `yong-hu` |
| 大写英文 | 转换为小写 | `OAuth` → `oauth` |
| 数字 | 保持原样 | `2.0` → `2-0` |
| 特殊字符 | 替换为连字符 | `@#$` → `-` |
| 空格 | 替换为连字符 | ` ` → `-` |
| 连续连字符 | 合并为单个 | `--` → `-` |
| 首尾连字符 | 去除 | `-abc-` → `abc` |
