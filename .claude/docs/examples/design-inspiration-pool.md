# Design Inspiration Pool

> 80+ 位艺术家/设计师/建筑师灵感来源池
> 用于 UI_PROTOTYPE_TEMPLATE.md Phase 2: Design Inspiration Sampling

---

## 使用方法

从不同类别中随机采样 2 位大师，生成风格转译说明。

**采样策略**:
- 从不同类别中选择（避免风格过于相似）
- 优先组合：现代主义 + 生成艺术，建筑 + 平面设计
- 避免组合：同类别内的大师（如两位都是建筑师）

**转译模板**:
```
**灵感来源X**: {艺术家姓名} ({类别})
**核心特质**: {艺术家代表性特点,1-2句话}
**网页转译**:
  - 色彩: {具体色值} → {应用场景}
  - 版式: {布局策略} → {CSS实现方式}
  - 形态: {视觉元素} → {HTML/CSS实现}
  - 动效: {动态特征} → {CSS/JS实现}
**禁止模仿**: 不复刻{具体作品名称}的{具体元素}
```
